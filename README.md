# TaskFlow - Flutter Task Manager App

TaskFlow is a simple task manager built with Flutter.

## Features
- Login screen
- Add tasks
- Delete tasks
- Multiple screens
- State management using setState()

## Screens
1. Login Screen
2. Task List Screen
3. Add Task Screen
